Grammar Documentation

Grammar

This document explains how the Grammar module works inside English Platform. It covers the learning path, exercise types, adaptive feedback, speech practice, progress persistence, and the module's links with vocabulary, global review, and error tracking.

Module Goal

Grammar works as the pedagogical backbone of the platform. It combines a guided path, mixed exercise formats, speaking practice, compact feedback, and adaptive reinforcement so grammar study feels active instead of static.

Level-based path Mixed exercises Microphone speaking Adaptive review Vocabulary integration

Exercise Types

  • Multiple choice: select the correct translation or structure.
  • Listening choice: hear a phrase and pick the right answer.
  • Cloze: complete the missing part of a sentence.
  • Dictation: listen and type the expected phrase.
  • Error fixing: identify and repair the wrong fragment.
  • Speech: speak the target sentence and receive phonetic feedback.

Lesson Flow

  • 1. Unit entry: the learner opens a unit from the main path.
  • 2. Unit guide: the concept summary and quick objective are shown.
  • 3. Exercise run: questions advance with progress bar, hearts, and manual audio.
  • 4. Adaptive reinforcement: missed questions can return later in the same session.
  • 5. Completion: the final screen summarizes the unit and updates the path.

Pedagogical Feedback

  • Compact: feedback no longer takes over half of the screen.
  • Contextual: it explains why the correct answer is correct.
  • Granular: it distinguishes grammar, lexical, order, and phonetic issues.
  • Conditional: error type and problematic excerpt only appear when the learner is wrong.
  • Speech scoring: the speaking exercise reports pronunciation, completeness, and rhythm.

Module Integrations

  • My Vocabulary: important words and error-related words can be saved automatically.
  • Error Notebook: recurring error patterns are stored for future reinforcement.
  • Global Review: fragile concepts can feed the global spaced-review queue.
  • Profile and reports: lesson outcomes contribute to the learner's broader progress view.

Implementation Notes

  • Theme color: #58cc02.
  • Typography: uses the same local DIN Round Pro family as the product.
  • Persistence: progress supports local fallback and authenticated cloud sync.
  • Purpose: this document is meant for product presentation and technical organization.
