Flashcards

This document explains the Flashcards module as a review engine for vocabulary and context. It covers the deck path, the SRS logic, the active review session, result metrics, and integration with My Vocabulary and the global spaced-review layer.

Module Goal

Flashcards was designed to review words and contexts quickly, with repeated exposure and priority-based reinforcement. The module combines deck progression, per-card SRS, difficult-card focus, and rich final reports.

Review Engine

  • Per-card states: new, learning, relearning, and review.
  • Ratings: easy, good, hard, and again influence the next interval.
  • Core memory fields: due date, interval, stability, difficulty, and forgetting index.
  • Only difficult mode: filters the session to focus on fragile cards.
  • Dynamic queue: missed cards can return later in the same session.

Session Flow

  • 1. Deck selection: the learner enters the path and opens an unlocked or difficult deck.
  • 2. Active review: the card can be flipped, listened to, sped up, and rated.
  • 3. SRS update: each response updates difficulty, stability, and the next review time.
  • 4. Vocabulary integration: reviewed words can feed My Vocabulary if the toggle is on.
  • 5. Final report: the learner sees accuracy, pace, total time, and fragile cards.

Results and Metrics

  • Total time: calculated from the first card to the last one.
  • Accuracy: final score with visual emphasis.
  • Average pace: cards reviewed per minute.
  • Priority reviews: cards that still need reinforcement.
  • Deck history: sessions, best accuracy, average performance, and latest review.

Module Integrations

  • My Vocabulary: ON/OFF toggle for sending reviewed words into the learner vocabulary.
  • Global review: fragile cards can be added to the cross-module review queue.
  • Profile and reports: session outputs enrich the learner's consolidated analytics.

Path Controls

  • Review schedule: decks show due cards and the next review window.
  • Done today: finished decks stay locked until the next daily cycle.
  • Only difficult cards: quick switch for reinforcement mode.
  • Vocabulary sync toggle: visible control for My Vocabulary updates.
