# Checklist do Grammar A1-C2

Atualizado em: 2026-03-30

## Objetivo

Definir tudo o que o modulo `Grammar` precisa ter de `A1` ate `C2` para virar uma trilha robusta, escalavel e comparavel a uma plataforma grande de idiomas:

- conteudo por nivel
- unidades e licoes
- progressao pedagogica
- tipos de exercicio
- revisao e reforco
- feedback
- audio e fala
- cobertura multilocale
- criterios de aceite por nivel

## Definicao de Pronto do Modulo

O `Grammar` so pode ser considerado completo quando:

- [ ] existe trilha completa de `A1` a `C2`
- [ ] cada nivel tem unidades suficientes para nao parecer so demonstracao
- [ ] cada unidade tem intro, licoes, checkpoints, revisao e tela final
- [ ] cada topico tem exemplos, contraste de erro comum e aplicacao pratica
- [ ] cada nivel mistura gramatica, vocabulario funcional, leitura curta, audio e fala
- [ ] o conteudo funciona com os tipos de exercicio do app
- [ ] existe revisao adaptativa por erro e por conceito
- [x] o backend consegue servir conteudo relacional por nivel, unidade, licao e idioma
